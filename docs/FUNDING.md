# FUNDING

## What you need

- A funded Polygon wallet for signing orders (your trading account)
- Sufficient USDC (or the token your wallet uses for fees/exposure)
- A target Polymarket market (`MARKET_ID`) that is active

## Safety first

- Start with small size / low exposure caps in `.env`
- Watch logs and metrics for the first run
- Keep real `PRIVATE_KEY` out of git commits
